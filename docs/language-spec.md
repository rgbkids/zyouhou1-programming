# info1 言語仕様

情報Iブラウザインタプリタで動く教育用言語 **info1** の仕様です。

## 設計方針

- Python 風の構文を採用（高校生が学習済みのため）
- **インデントベースのブロック構造**（4スペースまたは1タブ）
- 実装を複雑にする機能は削除
- 学習用サンプルが簡潔に書けることを最優先

---

## ブロック構造

インデントでブロックを表します。

```
if x > 5:
    print("大きい")
else:
    print("小さい")
```

- インデントはスペース4つ、またはタブ1つ
- ブロックは必ず1文以上の文を含む
- `pass` で空ブロックを表現可能

## 演算子

### 算術演算子

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `+` | 加算 / 文字列連結 | `1 + 2`, `"a" + "b"` |
| `-` | 減算 | `5 - 3` |
| `*` | 乗算 | `2 * 3` |
| `/` | 除算（浮動小数点） | `7 / 2` → `3.5` |
| `//` | 整数除算 | `7 // 2` → `3` |
| `%` | 余り | `7 % 2` → `1` |
| `**` | べき乗 | `2 ** 10` → `1024` |

### 比較演算子

| 演算子 | 意味 |
|--------|------|
| `==` | 等しい |
| `!=` | 等しくない |
| `<` | より小さい |
| `<=` | 以下 |
| `>` | より大きい |
| `>=` | 以上 |

### 論理演算子

| 演算子 | 意味 |
|--------|------|
| `and` | かつ |
| `or` | または |
| `not` | でない |

## 文

### 代入文

```
x = 1
a[0] = 99
```

### 条件分岐

```
if x > 0:
    print("正")
elif x == 0:
    print("ゼロ")
else:
    print("負")
```

### for 反復

```
for i in range(1, 6, 1):
    print(i)
```

`range(stop)`, `range(start, stop)`, `range(start, stop, step)` が使えます。

リストの反復も可能です。

```
scores = [80, 90, 70]
for s in scores:
    print(s)
```

### while 反復

```
x = 10
while x > 0:
    print(x)
    x = x - 1
```

無限ループ防止のため、最大 **100,000 ステップ** のガードがあります。

### break / continue

```
for i in range(0, 10, 1):
    if i == 5:
        break
    print(i)
```

### 関数定義

```
def add(a, b):
    return a + b

result = add(3, 4)
print(result)
```

### pass

```
def todo():
    pass
```

## 組み込み関数

| 関数 | 説明 | 例 |
|------|------|-----|
| `print(...)` | 標準出力 | `print("hello", x)` |
| `len(x)` | リスト/文字列の長さ | `len([1,2,3])` → `3` |
| `range(stop)` | 0〜stop-1 の整数リスト | `range(5)` → `[0,1,2,3,4]` |
| `range(start, stop)` | start〜stop-1 | `range(1, 4)` → `[1,2,3]` |
| `range(start, stop, step)` | ステップ付き | `range(0, 10, 2)` → `[0,2,4,6,8]` |
| `int(x)` | 整数変換 | `int(3.7)` → `3` |
| `float(x)` | 浮動小数点変換 | `float("3.14")` → `3.14` |
| `str(x)` | 文字列変換 | `str(42)` → `"42"` |
| `abs(x)` | 絶対値 | `abs(-5)` → `5` |
| `input(prompt)` | 標準入力（mock） | `x = input("名前: ")` |

## 組み込みモジュール

### random モジュール

```
import random   # 宣言不要・自動でロード済み

n = random.randrange(6)        # 0〜5 の乱数
n = random.randrange(1, 7)     # 1〜6 の乱数
n = random.randrange(0, 100, 2) # 0〜98 の偶数乱数
```

### webapi モジュール

```
data = webapi.get_json("zipcode", {"code": "1000001"})
print(data)
```

allowlist に登録された API のみ呼び出し可能です。

### device モジュール（模擬）

```
x = device.accelerometer_x()   # 加速度センサ値
t = device.temperature()        # 温度センサ値
device.led_show("Hello")        # LED に文字表示
device.led_clear()              # LED クリア
device.motor_set(50)            # モータ速度設定
```

## 演算子優先順位（高い順）

1. `**`
2. 単項 `-`, `not`
3. `*`, `/`, `//`, `%`
4. `+`, `-`
5. `==`, `!=`, `<`, `<=`, `>`, `>=`
6. `not`
7. `and`
8. `or`

## 制約事項

- クラス定義は非対応
- 例外処理（try/except）は非対応
- import 文は非対応（モジュールは自動ロード済み）
- ジェネレータ・内包表記は非対応
- 多重代入（`a, b = 1, 2`）は非対応
