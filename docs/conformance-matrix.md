# 情報I 適合性マトリクス

情報Iの各学習内容を、ブラウザインタプリタで実現する機能へ写像したマトリクスです。

## 分類凡例

| 分類 | 意味 |
|------|------|
| **言語機能** | インタプリタのコア言語として実装する構文・型・組み込み関数 |
| **教材UI** | 説明パネル・可視化・トレース表示など補助的なUI機能 |
| **模擬API** | ブラウザ上でシミュレートするAPI（実機不要） |
| **サンプル** | `.info1` ファイルとして提供する教材プログラム |

## 優先度凡例

| 優先度 | 意味 |
|--------|------|
| **Must** | 必須（情報I対応の最小セット） |
| **Should** | 推奨（授業で見せたい） |
| **Optional** | 任意（余裕があれば） |

## 実装対象外

| 項目 | 理由 |
|------|------|
| 実機デバイス連携（USB/BLE） | ブラウザ制約・授業設定の多様性のため模擬で代替 |
| ファイルI/O | ブラウザ環境のため対象外 |
| 外部ネットワーク（allowlist外） | セキュリティのため遮断 |
| マルチスレッド | Web Worker分離は v2 以降 |
| Python標準ライブラリ全般 | 教育用サブセット言語として再設計 |
