# 情報I 適合性マトリクス

## 概要
情報Iの要求項目をブラウザインタプリタで実現する形式に写像したマトリクス。

## 分類凡例
- **言語機能** … インタプリタの構文・実行エンジンで実現
- **教材UI** … UI パネル・可視化で実現
- **模擬API** … 許可済みAPI または モックで実現
- **サンプル** … `.info1` サンプルプログラムとして提供
- **M** = Must（必須）/ **S** = Should（推奨）/ **O** = Optional（任意）

## ブラウザ直接実現困難な項目の代替方針

| 項目 | 困難な理由 | 代替方針 |
|------|-----------|---------|
| 実機 micro:bit 連携 | ブラウザから直接 USB/BLE 制御困難 | `device.*` 模擬 API + UI パネル |
| 任意 URL へのHTTP通信 | CORS, セキュリティポリシー | allowlist 方式 + mock レスポンス |
| ファイルI/O | ブラウザサンドボックス | 対象外（テキスト入出力で代替） |
| バイナリ・低レベル演算 | 教育スコープ外 | 対象外 |
