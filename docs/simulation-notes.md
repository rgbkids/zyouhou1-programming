# モデル化とシミュレーション 解説ノート

## モデル化とは

現実の現象を、コンピュータで扱えるように**単純化・抽象化**することです。

| 現実の現象 | モデル |
|-----------|-------|
| サイコロを振る | `random.randrange(1, 7)` |
| コインを投げる | `random.randrange(0, 2)` |
| 温度変化 | スライダー値の変動 |

## シミュレーションの構成要素

```
モデル定義 → 初期状態設定 → 試行の繰り返し → 集計 → 考察
```

### サイコロの例

```python
# 1. モデル: サイコロ = 1〜6の乱数
# 2. 初期状態: 各目の出現回数 = 0
counts = [0, 0, 0, 0, 0, 0]

# 3. 試行: n回繰り返す
for i in range(0, n, 1):
    d = random.randrange(1, 7)
    counts[d - 1] = counts[d - 1] + 1

# 4. 集計: 割合を表示
for i in range(0, 6, 1):
    print(i + 1, ":", counts[i] / n * 100, "%")
```

## 大数の法則

試行回数を増やすほど、実験的な確率（相対頻度）は理論的な確率に近づきます。

| 試行回数 | 表の割合（コイン） |
|---------|-----------------|
| 10 回   | 20〜80% の振れ幅 |
| 100 回  | 40〜60% 程度 |
| 1000 回 | 48〜52% 程度 |

## 授業での活用ポイント

1. **モデルの限界を理解する**  
   サイコロの目は完全にランダムとは限らない。モデルは近似である。

2. **試行回数と精度のトレードオフ**  
   多く試行するほど正確だが、計算時間がかかる。

3. **シミュレーションの応用**  
   天気予報、感染症の広がり、株価予測など、現実でも使われている。
